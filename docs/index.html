Duplicate remover

Pobierz ostatnią wersję (Windows x64).

Program do usuwania duplikatów plików. Plik .exe należy umieścić w katalogu w którym zamierzamy skasować duplikaty. Program usunie duplikaty z katalogu i wszystkich podkatalogów.

Po zakończeniu działania programu lista skasowanych plików jest zapisywana do pliku logs.txt.
